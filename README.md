# 🐎 POLO — Project Management App

A Trello-inspired project management web application built with a modern fullstack JavaScript stack.

🔗 **Link:** [poloweb.vercel.app](https://poloweb.vercel.app)

---

## ✨ Features

- 🔐 **Authentication** — Secure user registration and login
- 📋 **Boards** — Create and organize boards by category (Work, Marketing, Personal, etc.)
- 📂 **Sections** — Add custom sections within each board to structure your workflow
- ✅ **Tasks** — Create tasks with:
  - Title and description
  - Color label
  - Expiration date with overdue indicator
  - Completion status
  - Member assignment
- 👥 **Team collaboration** — Add members to boards and assign tasks to them
- 🔔 **Notifications** — Get notified on relevant activity

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, Vite, Material UI |
| Backend | Node.js, Express |
| Database | PostgreSQL |
| Deployment | Vercel (frontend) |
